📌 Overview

Fake News Detection using AI is a Machine Learning and Natural Language Processing (NLP) based project designed to identify whether a news article is Real or Fake.
The system analyzes textual content from news articles and predicts authenticity using advanced AI techniques.This project helps reduce the spread of misinformation on digital platforms by automatically classifying news content with high accuracy.

🚀 Features

->Detects fake and real news articles

->NLP-based text preprocessing

->Machine Learning / Deep Learning classification

->User-friendly prediction system

->High accuracy text classification

->Easy deployment using Flask and Google Cloud


🧠Technologies Used

->Python

->Machine Learning

->Deep Learning

->NLP (Natural Language Processing)

->TensorFlow / Scikit-learn

->Pandas & NumPy
